# SDN Mininet Project

## Description
This project demonstrates Software Defined Networking using Mininet and Ryu controller.

## Topology
- 2 Hosts (h1, h2)
- 1 Switch (s1)

## Steps
1. Start Ryu controller
2. Start Mininet
3. Run pingall
4. Run iperf
5. Check flow table

## Results
- Ping successful
- High bandwidth observed
- Flow rules dynamically installed

## Conclusion
Demonstrates controller-switch interaction and SDN behavior.
